Tips for your group of 13
🚐
Hire a 16-seater minibus from Singapore for the full trip — easiest way to keep the whole group together.
🎟️
Book Lost World of Tambun tickets on Klook in advance. Group discounts available. Park is closed on Tuesdays!
🛶
Tasik Cermin is best in the morning for the mirror reflection effect. Queue early — it fills up fast on weekends.
🧓
Kek Lok Tong Cave Temple is flat and beautifully maintained — perfect for elderly members of the group.
🍽️
Call ahead to reserve for 13 pax — especially for bean sprout chicken at Lou Wong or Onn Kee restaurant.
🏨
For the city stay (Days 3–5), WEIL Hotel is a great pick — 4.5-star, central, attached to Ipoh Parade mall.

How to connect to Google Sheets (one-time setup)
  1. Open Google Sheets and create a new spreadsheet. Add these column headers in row 1:
    Timestamp | First Name | Last Name | Birthdate | Gender | Passport No | Date of Issue | Expiry Date
  2. Click Extensions → Apps Script in your Google Sheet.
  3. Delete any existing code and paste the Apps Script code below.
  4. Click Save, then Deploy → New deployment.
  5. Set type to Web app, set "Who has access" to Anyone, then click Deploy.
  6. Copy the Web app URL you receive.
  7. In this HTML file, find https://script.google.com/macros/s/AKfycbyrEZipMZ0YlbMjGM2Ph_Vj3VbJCrIi0AREn1rJ8ipBruvonxlDMe0AG28dOQ88SAH4bw/exec and replace it with your URL.
  8. Save and re-upload to Netlify. Done! ✅
Google Apps Script — paste this into Extensions → Apps Script
function doPost(e) {
  var sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();

  var data = JSON.parse(e.postData.contents);

  sheet.appendRow([
    new Date(),
    data.firstName,
    data.lastName,
    data.birthdate,
    data.gender,
    data.passportNo,
    data.issueDate,
    data.expiryDate
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({ result: "success" }))
    .setMimeType(ContentService.MimeType.JSON);
}
